Cereal Killer

Collaborated With

  • Connor Bell
  • JD Hoglan

Cereal Killer is a web tower defense game built in the Unity engine. This game was designed using the unity editor and written in C#. This project was written as a submission to the LudumDare competition. The LudumDare competition is a weekend-long competition where game developers create a game in response to a theme. This submission was in response to the theme "an unconventional weapon," in this case milk! The game plays similarly to other tower defense games, showing the path the bowls will take in green and allowing users to place milk towers with different effects in a grid world to prevent them from reaching the end.

Java SQLite Interface


While working on a databases project in Hendrix's Scalable Software class, I began to notice I was writing SQL queries which were largely reptetive. The natural solution: write a library to do it for me! This project is designed to be a tool to allow developers using Java and SQLite to quickly write applications without having to manually manage tables. Most notably, it posseses the ability to automatically save and query the database using user defined fields, generate tables automatically by reflecting over provided classes, and perform basic database maintenance without the developers intervention. This project is still under development - next is automated migrations between database schemas!

Logo Interpreter


The Logo language is designed to direct a 'turtle' around the screen. It allows users to specify instructions such as pen-up/down, left, right, forward, and backward. In our scalable software class, we designed Logo interpreters using the Gramblr parsing engine. This interpreter is a GUI for writing Logo code and interacting with the turtle.
